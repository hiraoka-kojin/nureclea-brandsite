# ヌレクリア ブランドサイト - 公開用ファイル

## 📁 ファイル構成

```
public/
├── index.html          # トップページ
├── style.css           # スタイルシート
├── tokusho.html        # 特商法表記ページ
└── images/
    └── product/
        └── main.jpg    # 商品のメイン画像（要配置）
```

## 🖼️ 画像の配置

`images/product/` フォルダに以下の画像を配置してください：

- **main.jpg** - 商品のメイン画像（必須）
  - 推奨サイズ: 横幅 800〜1200px
  - 形式: JPG または PNG
  - 背景: 白背景推奨

## 🚀 ローカルプレビュー

```bash
# 簡易サーバーで確認
python3 -m http.server 8000
# または
npx serve .
```

ブラウザで `http://localhost:8000` を開く

## 📦 本番デプロイ（Cloudflare Pages）

### 方法1: Gitリポジトリ連携（推奨）
1. GitHubにプッシュ
2. Cloudflare Pagesのダッシュボードでリポジトリを連携
3. ビルド設定:
   - ビルドコマンド: （なし）
   - 出力ディレクトリ: `public`

### 方法2: Wrangler CLIで直接デプロイ
```bash
npx wrangler pages deploy public --project-name nureclea
```
