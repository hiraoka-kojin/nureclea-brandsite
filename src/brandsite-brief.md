# ヌレクリア｜ブランドサイト要件設計

**作成日**: 2026-04-13
**依頼元**: strategy/
**制作担当**: brandsite/（新規プロジェクトとして別立て）

---

## 目的

ヌレクリアのブランドの顔となるサイトを作る。

- ブランド名で検索した人が「ちゃんとした商品だ」と確認できる
- DMを受け取った人が「怪しくない」と安心できる
- Amazon商品ページだけでは伝えきれないブランドの世界観を補完する
- SNSプロフィールのリンク先としても機能する

## ページ構成

### 1. ファーストビュー

- キャッチコピー + サブコピー
- 商品写真（メイン1枚）
- 「何のための商品か」が一目でわかること

**コピー案:**
> わたしの感覚を、もっと自由に。
> 女性のためのフェムケアクリーム「ヌレクリア」

### 2. こんな方へ（共感セクション）

ターゲットが「自分のことだ」と感じるセクション。

- なんとなく物足りなさを感じている
- 自分の体のことをもっと知りたい
- ケアの方法がわからない

※直接的な性的表現は避ける。「体と向き合う」「自分を大切にする」の文脈で

### 3. 商品の特徴

フェムケアクリームとしてどんな商品かを伝える。3つ程度。

- 例: デリケートゾーン専用 / 国内製造 / べたつかないテクスチャー
- 使い方の簡単な説明（塗るだけ、1日1回など）
- ※薬機法に注意。「効く」「治る」「感度が上がる」は不可
- 使用感の描写はOK（「しっとり」「じんわり温かい」など）

### 4. こだわり・成分

- 主要成分を2〜3個ピックアップ（成分名 + ひとこと説明）
- 安全性（無添加、国内製造など該当するもの）
- 「○○を配合」はOK、「○○が効く」はNG

### 5. ブランドストーリー

- 誰がなぜ作ったか（2〜3行でOK）
- 「女性が自分の体と向き合うきっかけを作りたい」的な文脈
- 堅くなりすぎず、共感できるトーンで

### 6. 購入方法

- Amazon商品ページへのリンクボタン
- 「Amazonでお求めいただけます」程度のシンプルな案内

### 7. モニター募集（期間限定セクション）

- 「現在、商品モニターを募集しています」
- 内容の簡単な説明:
  - 商品を無料でお届け
  - 1〜2週間お試し後にアンケート（5問）にご回答
  - お礼にAmazonギフト券（1,000円分）をお送りします
  - SNS投稿は任意
- 「XのDMからお声がけしています」と明記
- 募集が終わったらこのセクションを非表示にする

### 8. フッター

- ブランド名「ヌレクリア」
- お問い合わせ先（メールアドレス or Xアカウントへのリンク）
- 特定商取引法に基づく表記（別ページ or ページ下部に折りたたみ）

## デザインの方向性

| 項目 | 方向性 |
|------|--------|
| トーン | 清潔感・上品。アダルト感を出さない |
| 配色 | 白ベース + 淡いピンク or ラベンダー系のアクセント |
| フォント | 丸みのあるゴシック（読みやすさ優先） |
| 写真 | 商品写真を中心。人物写真は不要 |
| 全体の印象 | 「フェムケアブランド」として自然に見えること |

## 用意が必要な素材

| 素材 | 状況 | 備考 |
|------|------|------|
| 商品写真（正面） | 用意できる | 白背景推奨 |
| 商品写真（使用イメージ） | あれば望ましい | なくても可 |
| ブランドストーリー文 | 未作成 | strategy/ でドラフト or brandsite/ で作成 |
| 成分情報 | Amazon出品時の情報を流用 | |
| 特商法表記の内容 | 用意が必要 | 販売者名・住所・連絡先 |
| Xアカウント情報 | 既存（sns/ops/ で運用中） | リンク先として使用 |
| お問い合わせ用メールアドレス | 用意が必要 | |

## 注意事項

- 薬機法に抵触する表現は使わない
- 特商法表記は法律上の義務。省略しない
- モニター募集セクションは、募集終了後に非表示にできる構成にする
- SNSプロフィールのリンク先としても使うことを想定（ファーストビューの印象が大事）
