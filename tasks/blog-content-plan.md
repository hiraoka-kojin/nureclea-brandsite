## SEO・LLMO最適化チェックリスト

### 各記事共通で実施すること

**SEO対策：**
- [ ] タイトルタグに主要キーワードを含める（60文字以内）
- [ ] メタディスクリプションに検索意図を反映（120〜160文字）
- [ ] H2見出しにキーワードを自然に含める
- [ ] 画像にalt属性を設定
- [ ] 内部リンク（商品ページ、他記事へのリンク）
- [ ] 外部リンク（信頼できる医療機関等）
- [ ] 読みやすい文章構成（3〜5行で段落分け）

**LLMO対策：**
- [ ] 質問形式の見出し（H2/H3）を含める
- [ ] FAQセクションを設置
- [ ] 簡潔で明確な回答を提供
- [ ] リスト形式で情報を整理
- [ ] ステップバイステップの説明
- [ ] 「〜とは」「〜する方法」などの定義・解説を含める

**E-E-A-T向上：**
- [ ] 専門的な情報源を引用
- [ ] 医学的根拠を提示
- [ ] 実践的なアドバイス
- [ ] 更新日を明記
- [ ] 著者情報（ブランド情報）
